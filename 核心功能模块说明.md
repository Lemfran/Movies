# 核心功能模块说明

## 一、用户管理模块

### 1.1 功能概述
用户管理模块负责系统用户的注册、登录、权限验证及用户信息维护，是整个系统的基础模块。

### 1.2 核心功能
| 功能点 | 说明 |
|--------|------|
| 用户注册 | 新用户填写用户名、密码进行注册，系统自动分配用户ID |
| 用户登录 | 验证用户名密码，登录成功后创建Session会话 |
| 权限管理 | 区分普通用户和管理员权限，管理员可访问管理后台 |
| 会话管理 | 使用Session存储用户登录状态，避免重复传递用户ID |

### 1.3 技术实现
- **Controller**: `LoginController` 处理登录/注册请求
- **Service**: `UserService` 实现用户业务逻辑
- **DAO**: `UserDao` 负责用户数据访问
- **Interceptor**: `LoginInterceptor` 拦截未登录请求

---

## 二、电影管理模块

### 2.1 功能概述
电影管理模块负责电影信息的展示、查询、添加、修改和删除，是系统的核心数据模块。

### 2.2 核心功能
| 功能点 | 说明 |
|--------|------|
| 电影列表 | 展示所有电影信息，支持海报展示和基本信息查看 |
| 电影详情 | 显示单部电影的详细信息，包括简介、导演、演员等 |
| 添加电影 | 管理员可添加新电影，上传海报和填写详细信息 |
| 编辑电影 | 管理员可修改已有电影的信息 |
| 删除电影 | 管理员可删除电影及其相关评论 |

### 2.3 技术实现
- **Controller**: `MovieListController`(列表)、`MovieController`(详情)
- **Service**: `MovieService` 处理电影业务逻辑
- **DAO**: `MovieDao` 访问电影数据
- **数据库**: `movies` 表存储电影信息

---

## 三、评论管理模块

### 3.1 功能概述
评论管理模块实现用户对电影的评分和评论功能，支持评论的增删改查。

### 3.2 核心功能
| 功能点 | 说明 |
|--------|------|
| 发表评论 | 登录用户可对电影进行评分（1-5星）和撰写评论 |
| 评论列表 | 展示电影的所有评论，按时间倒序排列 |
| 我的评论 | 用户查看自己发表的所有评论 |
| 删除评论 | 用户可删除自己的评论，管理员可删除任意评论 |

### 3.3 技术实现
- **Controller**: `ReviewController` 处理评论相关请求
- **Service**: `ReviewService` 实现评论业务逻辑
- **DAO**: `ReviewDao` 访问评论数据
- **数据库**: `reviews` 表存储评论信息

---

## 四、系统管理模块

### 4.1 功能概述
系统管理模块为管理员提供后台管理功能，包括用户管理、电影管理、评论管理和系统日志查看。

### 4.2 核心功能
| 功能点 | 说明 |
|--------|------|
| 用户管理 | 查看所有用户信息，可删除用户账号 |
| 电影管理 | 管理电影信息，支持增删改查 |
| 评论管理 | 查看和删除不当评论 |
| 系统日志 | 查看用户操作日志，支持审计追踪 |

### 4.3 技术实现
- **Controller**: `ManagerController` 处理管理请求
- **Service**: 复用 UserService、MovieService、ReviewService
- **AOP**: `MyAspect` 自动记录操作日志

---

## 五、日志记录模块

### 5.1 功能概述
日志记录模块通过AOP技术自动记录用户的所有操作行为，用于系统审计和问题追踪。

### 5.2 核心功能
| 功能点 | 说明 |
|--------|------|
| 自动记录 | 通过AOP切面自动拦截并记录Controller层方法调用 |
| 日志内容 | 记录操作用户、操作方法、操作时间、请求参数 |
| 日志查询 | 管理员可在后台查看所有操作日志 |

### 5.3 技术实现
- **AOP切面**: `MyAspect` 类实现日志记录
- **注解**: 使用 `@Aspect`、`@Around` 等Spring AOP注解
- **数据库**: `logs` 表存储日志信息
